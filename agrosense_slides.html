AgroSense

Real-time market prices, forecasts & verified buyers — in Hausa & English

Team: AgroSense — Demo for Nigeria AI Hackathon

90s demo: show price query, forecast stub, and verified-buyer lookup

Problem

Smallholder farmers lack timely price information and access to trustworthy buyers. This leads to exploitation and post-harvest losses.

Example: farmers in Northern Nigeria receive market prices late via intermediaries.

Solution (MVP)

  • Chat & voice bot in Hausa + English
  • Real-time local market prices + 7-day forecast stub
  • Verified buyer registry & contact method

Works on Web chat, SMS/WhatsApp (via Twilio), and serverless backend.

Architecture

Bot → Function App (orchestrator) → Cognitive Search + Azure OpenAI + Azure ML + CosmosDB → Channels

Tonight: local CSV fallback + prompt templates. Cloud: use Key Vault + Managed Identity for secrets.

Demo plan

  1. Ask: "How much is maize in Kano?" — show price + forecast + recommendation
  2. Ask: "Will price rise this week?" — show forecast percent
  3. Ask: "Show verified buyers for maize in Kano" — show buyer list

Impact & Roadmap

KPIs: active users, forecast error (MAE), % trades via verified buyers, farmer satisfaction.

Phase 1: chat + RAG. Phase 2: ML forecasts + voice (Hausa) + WhatsApp. Phase 3: satellite NDVI alerts, ledger for verified trades.
